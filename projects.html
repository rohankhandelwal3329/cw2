Project 1: AI Debate Partner

Overview

An AI-powered debate training tool that helps students improve their argumentation and debating skills through real-time voice conversations. The application uses Deepgram's Voice Agent API for natural voice interactions and provides detailed feedback on debate performance.

Key Features

  • Voice-based Interaction — Natural voice conversations powered by Deepgram's Voice Agent API
  • Structured Debate Flow — Topic selection, side picking, back-and-forth debate, and detailed feedback
  • Detailed Feedback — Comprehensive coaching with strengths, areas to improve, and recommendations
  • Student Login — Personalized experience with name, Panther ID, and email registration
  • Auto-disconnect — Automatically ends session after AI delivers feedback

Technologies Used: Node.js • Express • JavaScript • Deepgram Voice Agent API • Web Audio API • Vercel

Project 2: Telematics Insurance Risk Pricing

Overview

A telematics-based insurance proof of concept that demonstrates how driving behavior data can be used for dynamic risk pricing. The system collects and analyzes driving events to calculate personalized insurance premiums based on actual driving patterns.

Key Features

  • REST API — Full-featured API for telematics data ingestion and retrieval
  • Event Simulation — PowerShell scripts to simulate driving events for testing
  • Risk Calculation — Algorithm to assess driver risk based on behavior patterns
  • SQLite Database — Persistent storage for telematics events and driver profiles
  • Documentation — Comprehensive API docs and system overview

Technologies Used: Python • Flask • SQLite • REST API • PowerShell • Data Analysis

Project 3: ChatZ - Gen Z Chatbot

Overview

A Gen Z-styled chatbot powered by Google's Gemini API. The chatbot uses slang, emojis, and sassy replies to create an engaging conversation experience. Features a toggle between clean (family-friendly) and mildly spicy modes—never harmful content.

Key Features

  • Gen Z Tone — Slang, emojis, short and sassy replies
  • Censor Mode Toggle — ON = family-friendly; OFF = allows mild humor
  • Modern Dark UI — Colorful rounded bubbles, soft shadows
  • Multi-chat Support — New Chat button + chat history sidebar
  • Context Memory — Sends recent conversation history to Gemini
  • LocalStorage Persistence — Conversations saved in browser

Technologies Used: React (Vite) • Tailwind CSS • Node.js • Express • Google Gemini API

Project 4: Face Recognition Blurring System

Overview

A full-stack application for detecting and selectively blurring faces in videos using RetinaFace and face recognition algorithms. Users can upload videos, select which faces to blur, and download the processed result—perfect for privacy protection.

Key Features

  • Video Upload — Handle video file uploads for processing
  • Face Detection — Accurate detection using RetinaFace algorithm
  • Face Recognition — Match and identify faces across video frames
  • Selective Blurring — Choose which faces to blur while keeping others visible
  • Progress Tracking — Real-time API for monitoring processing status

Technologies Used: Python • Flask • React • RetinaFace • OpenCV • Computer Vision
